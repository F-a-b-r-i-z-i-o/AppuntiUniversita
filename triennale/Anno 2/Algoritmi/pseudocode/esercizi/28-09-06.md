# Esercizio 2

Dato un albero binario di ricerca T, descrivere un algoritmo ricorsivo per stampare tutte le chiavi comprese fra due interi dati a e b.

```javascript
function rico(A, p, r, a, b){
    if(p <= r){
        if(A[p] < b && A[p] > a){
            print(A[p])
        }

        rico(A, p*2, r, a, b)
        rico(A, p*2+1, r, a, b)
    }
}
```
