# Esercizio 2

Sia A un array ordinato di n interi compresi fra [0,1]. Determinare Sum(A)

```javascript
function go(A, n){
    ris = Somma_numeri(A, 1, A.n)

    if (A[ris] != 0){
        ris -= 1
    }

    return n - ris
}

function Somma_numeri(A, p, r){
    if(p < r){
        q = int((p+r)/2)

        if(A[q]+A[q+1] == 0){
           p =  Somma_numeri(A, q+1, r)
        }
        else{
            p = Somma_numeri(A, p, q)
        }
    }

    return p
}
```
